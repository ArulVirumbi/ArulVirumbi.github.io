Students Union Website

  • Category: Web Development
  • Project timeline: Oct 2021 - Jan 2022
  • Website URL: su.psgtech.ac.in

Revamped Students Union website of PSG College of Technology

The project entails improving and designing various pages on the frontend and developing feature enhancements on the backend. For frontend development, HTML, CSS, and JavaScript are utilized, whereas PHP and SQL are employed for backend development. Some of the backend features consist of a Captcha-based authentication system for the admin panel, a report generation tool for events, and an announcement feature that shows notices on the homepage.
